\section{Restricted Bipartition}

\noindent In order for ASTRAL to run in  polynomial time, we must construct a set $\mathcal{X}$ of bipartitions which will limit the search space for the true tree $\mathcal{T}$. This set must be polynomial in size and computed in polynomial time for it to be of any use. Moreover, we would like it to form a bipartition cover with high probability, so that statistical consistency of the model is still preserved. 

To this effect, one useful class of sets when it comes to building $\mathcal{X}$ are the leaf sets:
%
\[L_\alpha := \{\text{leaves }\ell : \alpha \in c(\ell)\}\]
%
\noindent Say a subset $L \subseteq S$ of leaves is \textit{contained} within a given bipartition $\phi := (A|B)$ if:
%
\[L_\alpha \subseteq A \qquad \text{or} \qquad L_\alpha \subseteq B\]
%
It is easy to see for nonhomoplastic cognates $\alpha \in \Sigma_N $, if $\alpha$ is born on the edge $e$ and $\phi_e = (A_e | B_e)$ is the bipartition this edge generates in $T$ then $L_\alpha$ is contained in $\phi_e$. Moreover, if no deaths of $\alpha$ occur after its born then in fact $L_\alpha = A$ or $L_\alpha = B$ (depending on which one is below $e$). Hence given collections $\{L^{(c)}_\alpha\}_{c \in C, \alpha \in \Sigma^{(c)}}$ for each of our given characters, the most trivial choice for the candidate bipartitions would be the set:
%
\[\mathcal{X}_1 := \{(L^{(c)}_\alpha|S-L_\alpha) : c \in C, \alpha \in \Sigma^{(c)}\}\]
%
\noindent Trivially $|\mathcal{X}_1| = O(|C|)$, so its often much smaller than the total $2^k$ number of possible bipartitions. It is easy to see under mild assumptions this alone is sufficient for ensuring statistical consistency, since with positive probability a given non-homoplastic $\alpha$ is born along an edge $e$ and does not die at all, which we will show below. However, it is clear that $\mathcal{X}_1$ is bad at generating "large" bipartitions. Namely, if $\phi_e = (A|B)$ it is easy to see that the probability a given character born in edge $e$ survives down to the leaves is exponentially small in $|A|$. Hence we propose a few heuristics for expanding the size of $\mathcal{X}_1$ so it has better coverage probabilities. A natural solution is to allow multiple states $\alpha$ to contribute:
%
\[\mathcal{X}_m := \left\{\bigcup_{i=1}^r L_{\alpha_i}^{(c)}  \;: \; c \in C, \alpha_i \in \Sigma^{(c)}, r \leq m\right\}\]
%
\noindent It is easy to see that $\mathcal{X}_n \subseteq \mathcal{X}_m$ for $n \leq m$, so in particular $\mathcal{X}_1 \subseteq \mathcal{X}_m$. Hence asymptotic consistency using $\mathcal{X}_m$ follows from that of $\mathcal{X}_1$. Moreover, $|X_m| = O(|C|^{m+1})$ so again we have some control on the size relative to the number of characters. \red{Should maybe add in some compatibility condition; e.g. only take the union of leaf sets that overlap. Recall that under the standard formulation of the Canby model we typically assume the birth rates $\lambda_i^{(c)}$ and death rates $\mu_i^{(c)}$ for a specific character are an immigration-death process:

\[\lambda_i^{(c)} = \lambda^{(c)}, \qquad \mu_i^{(c)} = \begin{cases}
    
i\mu^{(c)} & i > 1 \\
0 &  i = 1 \end{cases}\]

\noindent Namely, the birth rate is constant and the death rate is proportional to the current population size, except for the constraint that the population cannot die off. We will assume the model has these rates below. 

\begin{theorem}[Bipartition Cover Bounds for Canby With No Borrowing]
Let $\mathcal{B}$ be the set of bipartitions of a phylogenetic tree $\mathcal{T}$ with $k$ leaves and minimum/maximum branch lengths $T_{min}, T_{max}$ respectively. Suppose further we have a Canby model $\left\{\Sigma^{(c)}, \lambda^{(c)}, \mu^{(c)}, \rho^{(c)}\right\}_{c \in C}$ on $\mathcal{T}$ with no borrowing and whose rates follow an immigration-death process. Let:
%
\[C_N := \{c \in C \; : \; \nu^{(c)}(\Sigma_N^{(c)}) > 0\}\]
%
\noindent be the set of characters with positive chance of non-homoplasy. Furthermore, assume we have the following assumptions on the parameters of the Canby model:

\begin{enumerate}
    
    \item[(i)] \textit{Nonvanishing Birth Rates:} Suppose that $\lambda^{(c)} \nu^{(c)}(\Sigma_N^{(c)}) \geq b > 0$ for all $c \in C_N$.

    \item[(ii)] \textit{Bounded death rates}: Suppose $\mu^{(c)} \leq D < \infty$ for all $c \in C_N$. 
\end{enumerate}
%
\noindent Then we have exponential convergence of the coverage probability:
%
\[\P(\mathcal{B} \nsubseteq \mathcal{X}_1)  \leq\sum_{\ell=2}^{k-2} q_\ell^{|C_N|} \leq (k-3)q_{k-2}^{|C_N|}\qquad q_\ell := \exp\left(-bT_{min} \exp\left[-2\ell DT_{max}\right]\right) \]
%
\noindent In particular, we have asymptotic consistency when $|C_N| \to \infty$:
%
\[\lim_{|C_N| \to \infty} \P(\mathcal{B} \subseteq \mathcal{X}_1) = 1,\]
%
\noindent Moreover, \red{$ \E |\mathcal{X}_1| \leq \frac{\lambda}{\mu} k $ }
\label{thm:canby_bipartition_cover_bounds_no_borrowing}
\end{theorem}
%
In particular, this implies if we run the ASTRAL as proposed in the "Identifiability of Canby" paper, restricted to the bipartition set $\mathcal{X}_1$, this is a statistically consistent estimate of the true underlying phylogenetic tree $T$, and that we need $\exp(\Theta(k))$ characters to ensure recovery with high probability. Note that as the proof below makes clear, these assumptions are just some possible sufficient conditions. Really, we just require:
%
\[\sum_{c \in C} \lambda^{(c)} \nu^{(c)}(\Sigma_N^{(c)}) \exp\left(-2k \mu^{(c)} T_{max}\right) \to \infty\]
%
\noindent Namely, we need characters that are sufficiently likely to produce non-homoplastic states, and that these states are at least somewhat likely to survive until the leaves.

\begin{proof}[Proof of Theorem \ref{thm:canby_bipartition_cover_bounds_no_borrowing}]

Fix an edge $e \in E(\mathcal{T})$ with $k_e$ descendants and a given character $c$. For ease of notation, we drop all the $(c)$ superscripts for now. If $\phi_e$ is the corresponding bipartition, we will lower bound the probability that $\phi_e = (L_\alpha|S-L_\alpha)$ for some $\alpha \in \Sigma$.

First, note that as the birth-rate is constant the number $N_e$ of non-homoplastic states born along the edge $e$ satisfies:
%
\[N_e \sim \mathrm{Poisson}(\lambda T_e \cdot \nu(\Sigma_N)) \succeq_{st} Poisson(\eta) \qquad \eta := \lambda T_{min} \nu(\Sigma_N)\]
%
\noindent where $T_e$ is the length of the branch $e$ and $T_{min}$ the minimum branch length in the tree. 

For a given non-homoplastic $\alpha$ born along the edge $e$, we can lower bound the probability it has no deaths below $e$. By Lemma \ref{lem:death_rate_prob_bounds} the probability $\alpha$ survives along a branch of length $T$ is at least $\exp(-\mu T)$. There are $2k_e - 1 \leq 2k_e$ edges (including $e$) that $\alpha$ can die on. Each is at most length $T_{max}$, so we can lower bound the probability $\alpha$ has no deaths at all by:
%
\[\P(\alpha \text{ has no deaths} \; |\alpha \in \Sigma_N, \alpha \text{ born on e}) \geq \exp(- \tau_{k_e}) \qquad \tau_{k_e} := 2\mu k_e T_{max}\]
%
\noindent Using this result followed by $N_e \succeq_{st} \mathrm{Poisson}(\eta)$ we see:
%
\[\P(\exists\alpha \; \phi_e = (L_\alpha, S-L_\alpha)) \geq \E_{n \sim N_e} \left(1-(1-\exp(-\tau))^n\right) \geq 1-\exp(-\eta e^{-\tau})\]
%
 From here, we can essentially piggy-back off my bipartition cover paper for the multispecies coalescent model. Namely, the caterpillar bound applied to our $|C|$ characters implies:
%
\[\P(\mathcal{B} \subseteq \mathcal{X}_1) \geq 1 - \sum_{\ell=2}^{k-2} \prod_{c \in C}\exp\left(-\eta^{(c)} e^{-\tau_{\ell}^{(c)}}\right) =  1 - \sum_{\ell=2}^{k-2} \exp\left(-\sum_{c \in C}\eta^{(c)} e^{-\tau_{\ell}^{(c)}}\right)\]
%
\noindent Note none of the above uses our boundedness assumptions on the birth or death rates $\lambda^{(c)}, \mu^{(c)}$. Using these assumptions on the birth and death rates, we see:
%
\[\P(\mathcal{B} \subseteq \mathcal{X}_1) \geq 1 - \sum_{\ell=2}^{k-2}q_\ell^{|C_N|} \geq 1-(k-3)q_{k-2}^{|C_N|} \qquad q_\ell := \exp\left(-bT_{min} \exp\left[-2\ell DT_{max}\right]\right)\]
%
\noindent Hence we get the desired exponential convergence of the converge probability.

To bound $|\mathcal{X}_1|$, note that \red{we have roughly $\mathrm{Poisson(\frac{\lambda}{\mu}})$ states in each leaf, so we roughly have $\E |\mathcal{X}_1| \leq k \cdot \frac{\lambda}{\mu}$. This sort of implicitly assumes we don't have too many states present at the root, but of course this depends on $\rho$. }
\end{proof}

\subsubsection{Using Yule Trees}
\noindent Suppose a given state $\alpha \in \Sigma_N$ is born along an edge $e$. As we progress down the tree, every time we reach a branching point the number times $\alpha$ occurs in the current generation increases by one. When $\alpha$ dies in some branch, it decreases by one. 

Of course, this birth mechanism is completely deterministic and depends on the topology of the unknown species tree. But if we assume the underlying tree is itself generated via some random birth-process, we can explicitly analyze the expected fraction of leaves $\alpha$ reaches.

In particular, if we assume the underlying tree is a Yule tree with $\mathrm{Exp}(\gamma)$ branch lengths, then the number $A_t$ of $\alpha$ alive at a given point in time $t$ below the edge $e$ follows a birth-death process with birth and death rates $\gamma, \mu$ per individual respectively. It is easy to see:
%
\[\E A_t = e^{(\gamma - \mu)t}\]
%
\noindent If we let $N_t$ be the number of lineages in our yule tree, it is also easy to see:
%
\[E N_t = e^{\gamma t}\]
%
\noindent so it takes roughly $t_k := \gamma^{-1} \log(k)$ time to reach $k$ lineages. At this time, $\alpha$ occupies a fraction:
%
\[\frac{\E M_{t_k}}{\E N_{t_k}} = e^{-\mu t_k} = k^{-\mu/\gamma}\]
%
\noindent of the leaves approximately. This suggests choosing $\tau(k) \asymp 1- k^{-\mu/\gamma}$ if we hope to recover the given bipartition with decent probability.

\section{Sample-Complexity Bounds}

\noindent Assume we run the exact version of ASTRAL. We would like to know how many cognates are required to ensure we recover the true phylogenetic tree $T$ with high probability. To analyze this, suppose that we have independent characters $(c_i)_{i=1}^n$. 

We outline two different inference procedures. We will always assume the Canby model uses immigration-death rates. Our key tool will be a coupling of the Canby model to a standard immigration-death process (ID) run along the branches of our species tree $T^*$. It relies on two simple observations about how the set of nonhomoplastic states evolves over time:

\begin{itemize}
    \item Births always occur at rate at least $\lambda_N := \lambda \nu(\Sigma_N)$ and at most $\lambda$.

    \item Deaths always occur at rate at most $\mu$ per individual.
\end{itemize}
%
\noindent In particular, we can couple an ID process $(Y_s)_{s \in T^*}$ to our Canby $(X_s)_{s \in T^*}$ so that:
%
\[ Y_s \cap \Sigma_N \subseteq X_s \cap \Sigma_N, \quad \forall s \in T^*\]
%
\noindent In this way, we only reduce the number of nonhomoplastic states we observe in the leaves, which heuristically makes inference more challenging. 

To produce this coupling, use a single $\tau \sim \mathrm{\Exp}(\lambda)$ clock for each branch of $T^*$ to determine possible births. If $\tau$ rings at location $s \in T^*$, sample a new state $\alpha \notin X_{s^-}$ for the Canby model as usual, namely according to the conditional distribution:
%
\[\frac{\nu(\cdot \cap (\Sigma \setminus X_{s^-}))}{\nu( \Sigma \setminus X_{s^-})}\]
%
\noindent Hence we let $X_s = X_{s^-} \cup \{\alpha\}$. We update our ID process $Y_s$ as follows: if $\alpha \notin \Sigma_N$, then let $Y_s = Y_{s^-}$. Namely, ignore the homoplastic states. Otherwise:
%
\[Y_s = \begin{cases}
    Y_{s^-} \cup \{\alpha\} & \mathrm{w.p.}  \quad \nu(\Sigma \setminus X_{s^-})\\
    Y_{s^-} & \mathrm{w.p.} \quad 1-\nu(\Sigma \setminus X_{s^-}) \\
\end{cases}\]
%
\noindent This ensures that the birth rate in $(Y_s)$ along a given branch is always $\lambda_N$. 

Similarly, when a new state $\alpha$ is born in $X_s$, assign it an independent $\tau_\alpha \sim \mathrm{Exp(\mu)}$ clock. If it rings at location $s \in T^*$, let:
%
\[X_s = \begin{cases}
    X_{s^-} \setminus \{\alpha\} & X_{s-} \neq \{\alpha\}\\
    \{\alpha\} & \mathrm{otherwise}
\end{cases}\]
%
\noindent Then let $Y_s = Y_{s^-} \setminus \{\alpha\}$. Namely, $(X_s)$ simply ignores the deaths causing the state set to become empty, while $(Y_s)$ does not.

\subsection{Properties of Immigration-Death Processes}

Let $\mathrm{ID}(\theta, \mu, \lambda)$ denote an immigration-death process $Y_t$ with $|Y_0| \sim \mathrm{Poisson}(\theta)$, birth rate $\lambda$, and death rate $\mu$ per individual. We start by understanding the population size in such a process. 

\begin{lemma}[Population Sizes in ID Processes]
    If $(Y_t)_{t \geq 0} \sim \mathrm{ID}(\theta, \mu, \lambda)$ then: 
    %
    \[|Y_t| \sim \mathrm{Poisson}(m(t)), \quad m(t) = \theta e^{-\mu t} + \frac{\lambda}{\mu}(1-e^{-\mu t}).\]
    %
    \noindent In particular, its stationary distribution is $\mathrm{Poisson}(\lambda/\mu)$. Moreover for $s < t$:
    %
    \[\E|Y_t \cap Y_s| = m(s) \cdot  e^{-\mu (t-s)}\]
    %
    \label{lem:pop_sizes_in_ID_processes}
\end{lemma}

\begin{proof}
    Kolmogorov's forward equations tell us:
    %
    \[\frac{d}{dt}p_k = \lambda p_{k-1} + \mu (k+1) p_{k+1} - (k\mu + \lambda) p_{k}, \quad p_k(0) = \frac{e^{-\alpha} \alpha^k}{k!}\]
    %
\noindent It is easy to see the Poisson probabilities $p_k(t) =  \frac{e^{-m(t)}m(t)^k}{k!}$ satisfy these equations, and hence $X(t) \sim \mathrm{Poisson}(m(t))$. The covariance formula follows from the fact:
\[Y_t = \mathrm{Binomial}(X_s, e^{-\mu (t-s)}) + Z, \quad |Z| \sim \mathrm{Poisson}(m_0(t-s))\]
\noindent where these two terms are independent. The formula follows from the law of total covariance. 
\end{proof}

Since the dynamics of the ID process and Canby model essentially agree when there are more than one state, this reduces to studying the hitting times of our ID process.

\subsection{Canby Model}
\noindent Given a single path from the root of the species tree to some leaf, we can keep track of the number of states appearing along this path under the Canby model. This is a simple birth-death process $\{S_t\}_{t \geq 0}$ with birth rates $\lambda_i \equiv \lambda$ and death rates $\mu_i = i\mu \cdot 1\{i > 1\}$. Then:

\begin{lemma}
    The stationary distribution of $S_t$ is  $ \pi = (Z | Z > 0)$ for $Z \sim \mathrm{Poisson}(\lambda/\mu)$. 
    %
    \[\pi_i = \frac{(\lambda/\mu)^i}{i!(e^{\lambda/\mu}-1)}, \quad \forall i \geq 1.\]
\end{lemma}

\begin{proof}
    This process is reversible, so by detailed balanced:
    %
        \[\pi_{i+1}(i+1)\mu = \lambda \pi_i, \quad \forall i \geq 1\]
    %
    \noindent Recursively applying this shows $\pi_i \propto \frac{(\lambda/\mu)^i}{i!}$ as desired.
\end{proof}

\noindent The ergodic theorem tells us as $T \to \infty$ the fraction of the time $S_t$ spends in state 1 converges almost surely to $\pi_1 = \frac{m}{e^m-1}$. Heuristically, over long-branches our effective death-rate then becomes:
%
\[\mu_{\text{eff}} = \mu \left(1-\pi_1\right) = \mu \cdot \frac{e^m-m-1}{e^m-1}\]
%
\noindent If $\lambda \ll \mu$, then $\mu_{\text{eff}} \approx \frac{\lambda}{2} $ which is significantly less than $\mu$ in this low-polymorphism regime. If $\lambda \gg \mu$, then $\pi_1$ is exponentially small in $m$, so the process is rarely in state $1$. Hence it becomes well-approximated by our ID process coupling. 


For the following, we will also need two-sided bounds on the probability a given state $\alpha$ dies along a specific branch, which the next lemma provides.

\begin{lemma}[Survival and Death Bounds]

    Let $p_{n,T}$ be the probability a given state $\alpha$ dies along a branch of length $T$ in the Canby model, where we start the branch with $n$ states. Then:
    %
    \[\frac{1}{2} (1-e^{-\lambda T/2}) (1-e^{-\mu T}) =:\xi_{T}\leq p_{n,T} \leq 1-\eta_{T} := 1-e^{-\mu T}\]
    %

    \label{lem:death_rate_prob_bounds}
\end{lemma}

\noindent We will often be interested in applying these bounds to a specific path in the tree.     %
    \[\P(\alpha \text{ survives along branch} \mid \tau) = e^{-\mu T \cdot  \tau}\]
    %
    \noindent Hence the survival probability depends only on the Laplace transform $\phi$ of $\tau$:
    %
        \[\P(\alpha \text{ survives along branch}) = \phi(\mu T)\]
    %
    \red{For now we add two simple bounds, hopefully to improve later.} Trivially $\tau \leq 1$, which gives the upper bound. For the lower bound, note that $p_{n,T}$ is increasing in $n$ by a simple coupling argument. Hence WLOG we assume $n=1$.     \noindent An even simpler bound is:

    \[p_{n,T} \geq \frac{1}{2} (1-e^{-\lambda T/2}) (1-e^{-\mu T})\]

    \noindent which is the probability we have a birth of some state $\beta$ by time $T/2$, a death of either $\alpha, \beta$ in the remaining time $T/2$, and that this death is $\alpha$. 
\end{proof}

\noindent so in fact the correct topology is \textit{positively biased} for such $\alpha$. Again, the bias can only increase for all other $\alpha$, so overall there is a positive bias towards the correct topology when the induced subtree of $u,v,x,y$ is balanced.

 But where this construction fails is if the leaves $u,v,x,y$ induce a caterpillar tree, as we can no longer exploit the symmetry of the induced subtree. In the latter case, in a sense the "most probable topology" is not always $uv||xy$, and whether or not it is depends on the interior branch lengths and model parameters it seems. 

 In general, there are at least as many caterpillar induced subtrees as there are balanced ones and, depending on the underlying tree topology, often much more. So it is crucial that things work under this caterpillar tree topology.

Still, I wonder if this result is useful at all. 

\red{I wonder if we could get the even simpler:}

\[M'_{uv||xy} = \{\alpha \in \Sigma_N : \alpha \in (c(u) \cap c(v)) \cup (c(x) \cap c(y)) \}\]

\noindent to work.
